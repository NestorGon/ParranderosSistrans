Para ejecutar el proyecto se deben reemplazar las credenciales del archivo persistence.xml que se encuentra en
./src/main/resources/META-INF
Posterior a esto se debe ejecutar el script de CreacionEsquemaVacuAndes.sql y luego el de PoblarTablas.sql
Estos archivos se encuentran en ./docs
Una vez ejecutados estos scripts se puede ejecutar el proyecto.
Para esto se debe ejecutar la clase de InterfazVacuAndesApp.java y ya se puede interactuar con la aplicación
